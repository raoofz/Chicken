🧠 1. Smart Notes → Intelligent Actions (Core Feature)
Requirement:

Manager writes free-text notes like:

“انتبهوا على العلف اليوم لأنه قليل”
“في دجاج مريض تابعوه”
“نظفوا المكان جيداً”

System MUST:
Analyze the text
Understand meaning (not just keywords)
Extract:
Type	Example
Task	Clean area
Warning	Low feed
Alert	Sick chickens
Output:

Convert into structured data:

{
  type: "TASK" | "WARNING" | "ALERT",
  priority: "LOW" | "MEDIUM" | "HIGH",
  message: "...",
  relatedModule: "feed | health | environment"
}

📁 core/ai/noteInterpreter.ts

One note → multiple outputs
