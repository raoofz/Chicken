⚙️ 2. Service Worker (Advanced — Not Basic)

Implement a custom Service Worker with intelligent caching strategies.

🎯 Required Strategies:
1. App Shell → Cache First
HTML
CSS
JS
2. API Requests → Network First (with fallback to cache)
3. Images → Stale While Revalidate
🔧 Implementation Requirements:
Cache versioning system
Automatic cleanup of old caches
Offline fallback page (/offline)
Graceful failure handling
Optional: Background Sync for pending actions
Example Logic:
self.addEventListener('fetch', (event) => {
  const request = event.request;

  if (request.url.includes('/api/')) {
    event.respondWith(networkFirst(request));
  } else {
    event.respondWith(cacheFirst(request));
  }
});
